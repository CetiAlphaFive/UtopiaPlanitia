# Summarize LOCO Variable Importance

Prints a formatted summary of LOCO variable importance results.
Equivalent to calling [`print()`](https://rdrr.io/r/base/print.html) on
the object.

## Usage

``` r
# S3 method for class 'cf_loco'
summary(object, ...)
```

## Arguments

- object:

  An object of class `"cf_loco"` returned by
  [`cf_loco()`](https://cetialphafive.github.io/UtopiaPlanitia/reference/cf_loco.md).

- ...:

  Additional arguments (currently unused).

## Value

The `cf_loco` object (invisibly).
